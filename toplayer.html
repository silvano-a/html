We've all been there. The interface requires a modal, or a dialog to come and pop up over all the content. Preferably with a nice transition, the content in the background should be dimmed. The content stop scrolling.

First we have to create HTML elements to do this. It might look something like:

            
<div class="legacy-modal">
    <div class="modal-backdrop"></div>
    <div class="modal-content">
        <h2>Modal Title</h2>
        <p>This is the content of the modal.</p>
        <button id="close-modal">Close</button>
    </div>
</div>

Note that for this to work we would most likely add this at the bottom of the body, just like we've done on the HTML page you're viewing now.

The next step would be adding CSS to hide the modal initially, so we can show it later on with javascript.

            
.legacy-modal {
    display: none; /* Hidden by default */
    position: fixed; /* Stay in place */
    z-index: 9999; /* Sit on top */
    left: 0;
    top: 0;
    width: 100%; /* Full width */
    height: 100%; /* Full height */
    overflow: auto; /* Enable scroll if needed */

    .modal-backdrop {
        position: fixed;
        z-index: 1;
        top: 0;
        left: 0;
        width: 100%;
        height: 100%;
        background-color: rgba(0, 0, 0, 0.5); /* Semi-transparent background */
    }

    .modal-content {
        position: relative;
        z-index: 2;
        background-color: #fefefe;
        margin: 15% auto;
        padding: 20px;
        border: 1px solid #888;
        width: 80%;
        max-width: 600px;
        box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
        color: #3a3a3a;
    }
}

Now, let's add some javascript to make this work! we'll add the javascript on the end of this page in the body too, so you can test it out.

            
document.getElementById('open-modal').addEventListener('click', openModal);
document.getElementById('close-modal').addEventListener('click', closeModal);
document.getElementById('modal-backdrop').addEventListener('click', closeModal);

function openModal() {
    const modal = document.querySelector('#legacy-modal');
    modal.style.display = 'block';
}
function closeModal() {
    const modal = document.querySelector('#legacy-modal');
    modal.style.display = 'none';
}
